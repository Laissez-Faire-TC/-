# プロジェクト概要

こうせいテニスクラブ向け会員管理・遠征管理 Web アプリ（PHP MVC）。

---

## サーバーへの再アップロード対象

実装変更後にサーバーへアップロードが必要なフォルダ・ファイルを以下にまとめる。
**回答の末尾には必ず変更したファイルの一覧を記載すること。**

### アップロード対象フォルダ

| フォルダ / ファイル | 用途 |
|---|---|
| `src/Controllers/` | コントローラークラス（API・画面制御） |
| `src/Models/` | モデルクラス（DB操作） |
| `src/Services/` | サービスクラス（外部連携・ビジネスロジック） |
| `src/Core/` | フレームワーク中核（Router, Auth, Request, Response 等） |
| `views/` | PHP テンプレート（HTML + JS + CSS インライン） |
| `public/css/` | スタイルシート |
| `public/js/` | JavaScript ファイル |
| `index.php` | エントリーポイント（ルーティング定義） |
| `database/` | マイグレーション SQL（サーバーで手動実行が必要） |

### アップロード不要なもの

- `.claude/` — Claude Code のローカル設定
- `vendor/` — Composer 依存（サーバー側で `composer install` 済みのため）
- `.git/` — バージョン管理ディレクトリ

### マイグレーションが必要な場合

`database/` 以下の SQL ファイルをサーバーの DB に対して手動実行すること。
新しい SQL ファイルが追加された場合は必ずアップロード対象一覧に含める。
